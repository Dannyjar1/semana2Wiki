Wiki de Datos Abstractos

Pila

Una pila es una estructura de datos lineal que sigue el principio LIFO (Last In, First Out), es decir, el último elemento que se inserta es el primero en ser eliminado.

Las operaciones básicas en una pila son:

  • Push: agrega un elemento a la pila.
  • Pop: elimina el último elemento agregado a la pila.
  • Top: devuelve el último elemento agregado sin eliminarlo.


Cola

Una cola es una estructura de datos lineal que sigue el principio FIFO (First In, First Out), es decir, el primer elemento que se inserta es el primero en ser eliminado.

Las operaciones básicas en una cola son:

  • Enqueue: agrega un elemento a la cola.
  • Dequeue: elimina el primer elemento agregado a la cola.
  • Front: devuelve el primer elemento agregado sin eliminarlo.


Listas

Una lista abstracta define una colección ordenada de elementos, donde se pueden realizar operaciones como agregar elementos, eliminar elementos, buscar elementos, etc.

Algunos tipos de listas abstractas incluyen:

  • Listas enlazadas.
  • Listas doblemente enlazadas.
  • Listas circulares.


Grafos

Un grafo abstracto es una colección de nodos interconectados mediante aristas.

Los grafos abstractos se utilizan para representar relaciones y conexiones entre elementos en una variedad de aplicaciones como:

  • Redes sociales.
  • Sistemas de rutas.
  • Algoritmos de búsqueda.


Árboles

Un árbol abstracto es una estructura de datos jerárquica que consta de nodos interconectados. Los árboles abstractos se utilizan ampliamente en algoritmos de búsqueda, estructuras de datos balanceadas como los árboles binarios de búsqueda, y en representaciones de datos como los árboles de expresión.
